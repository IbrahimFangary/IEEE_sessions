# This is an IEEE Journal Club intro course on single-cell and spatial transcriptomics, organized into two main components:

### 1. Session 1 notebook (session_1_FigureOne.ipynb) — a hands-on walkthrough reproducing Figure 1 from a 2021 Nature Genetics paper that built a single-cell and spatial atlas of human breast cancers.
### 2. ST Race presentation — a conceptual overview of spatial transcriptomics technologies, covering the three major approaches (ISH/MERFISH, ISS, and ISC/Visium), key technical parameters (resolution, gene count, detection efficiency), data formats, analytical goals (spatial domains, cell type ID, deconvolution), and applications.
### 3. sketch analysis folder — covers data sketching techniques for handling large single-cell datasets efficiently in Seurat.


### 📄 Reference
Wu, S.Z., Al-Eryani, G., Roden, D.L. et al. A single-cell and spatially resolved atlas of human breast cancers. Nature Genetics 53, 1334–1347 (2021).
